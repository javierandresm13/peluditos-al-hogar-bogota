# 🐾 Pets Love Bogotá — Peluquería Canina a Domicilio

Sitio web profesional para **Pets Love**, servicio de peluquería canina y mascotas a domicilio en Bogotá.

## 🌐 Sitio en vivo

**Dominio genérico:** `https://javierandresm13.github.io/peluditos-al-hogar-bogota/`

## 📁 Estructura del proyecto

```
peluditos-al-hogar-bogota/
├── index.html              ← Landing page principal
├── admin/                  ← Panel de administración del dueño
│   ├── index.html          ← Login + dashboard
│   ├── admin.js            ← Lógica del panel admin
│   └── admin.css           ← Estilos del panel
├── assets/
│   ├── css/styles.css      ← Estilos del sitio web
│   ├── js/main.js          ← Lógica del frontend
│   └── images/             ← Carpeta para imágenes
└── data/
    ├── site.json           ← Contenido editable del sitio
    ├── services.json       ← Servicios ofrecidos
    ├── products.json       ← Productos de aseo canino
    ├── gallery.json        ← Galería de fotos
    └── loyalty.json        ← Planes de fidelidad
```

## 🔧 Panel de Administración

Accede al panel en: `/admin/`

**Contraseña por defecto:** `petslove2025`

### Lo que puedes editar:
- ✅ Títulos, subtítulos y textos de todas las secciones
- ✅ Servicios: agregar, editar, eliminar (con precios)
- ✅ Productos de aseo canino: catálogo completo
- ✅ Galería de fotos con URLs de imágenes
- ✅ Planes de fidelidad con beneficios
- ✅ Información de contacto (WhatsApp, Instagram, email)

### Cómo publicar los cambios:
1. Desde el panel Admin, ve a "Publicar cambios"
2. Da clic en "Descargar datos actualizados"
3. Sube los archivos JSON a la carpeta `/data/` del repositorio en GitHub
4. GitHub Pages publica automáticamente en minutos

## 🚀 Tecnologías

- **Frontend:** HTML5 + CSS3 + JavaScript vanilla
- **Admin panel:** Interfaz visual auto-gestionada
- **Almacenamiento:** Datos en JSON editables
- **Hosting:** GitHub Pages (100% gratuito)
- **Sin dependencias externas** — no requiere npm, build tools ni backend
