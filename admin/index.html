🚀 Publicar cambios en GitHub Pages

📋 Instrucciones paso a paso:

  1. Haz clic en "Descargar datos actualizados" para obtener los archivos JSON modificados.
  2. Ve a tu repositorio en github.com/javierandresm13/peluditos-al-hogar-bogota
  3. Sube los archivos descargados a la carpeta /data/ del repositorio.
  4. GitHub Pages publica automáticamente los cambios en unos minutos.

💡 Alternativa: Subir imágenes gratis

Para las fotos de la galería y productos, usa servicios gratuitos como: ImgBB, PostImage o Imgur. Luego pega la URL de la imagen en el editor de galería o productos.
