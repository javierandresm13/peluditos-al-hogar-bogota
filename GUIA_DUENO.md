## 🚀 PRIMERA VEZ: Conectar con GitHub (solo 1 vez)

Esto se hace **una sola vez** desde cualquier dispositivo (celular, tablet, PC).

### Paso 1: Entra al panel admin
Abre en tu navegador:
```
https://javierandresm13.github.io/peluditos-al-hogar-bogota/admin/
```
Contraseña: **petslove2025**

### Paso 2: Ve a "Publicar cambios"
En el menú de la izquierda, haz clic en **"Publicar cambios"**.

### Paso 3: Crea un token en GitHub
Verás una sección que dice "Conectar con GitHub". Sigue estos pasos:

1. Abre una nueva pestaña con este link:
   **https://github.com/settings/tokens**

2. Inicia sesión con el usuario: **javierandresm13**
   (Si no tienes la contraseña, pídesela a quien te entregó la página)

3. Haz clic en el botón **"Generate new token (classic)"**

4. Dale un nombre: **petslove-admin**

5. En "Scope" (permisos), marca solo: **repo** (se marcan todos los de repo automáticamente)

6. Haz clic en **"Generate token"** (abajo de la página)

7. ❗ **IMPORTANTE:** Aparecerá un código largo como este:
   ```
   ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
   **CÓPIALO AHORA** porque solo se ve una vez.

### Paso 4: Pega el token en el admin
1. Vuelve a la pestaña del panel admin
2. Pega el token en el campo que dice "Token de GitHub"
3. Haz clic en **"🔗 Conectar"**
4. Verás un mensaje verde: **"✅ Conectado"**

**¡Listo!** Ya puedes publicar cambios con 1 clic desde este dispositivo.
(Si usas otro celular o PC, repite solo el paso 2 y 4)

## ✏️ CÓMO EDITAR LA PÁGINA

Cada sección del menú de la izquierda tiene sus campos editables:

### 🏠 Hero (Portada)
- Cambia el título principal, el subtítulo y el texto del botón
- El botón rojo ya apunta al WhatsApp

### 📖 Nosotros
- Edita la descripción del negocio
- Cambia los destacados (uno por línea)

### 🛁 Servicios
- **Agregar servicio:** haz clic en "➕ Agregar servicio"
- **Cambiar precio:** edita el campo de precio
- **Eliminar:** haz clic en el botón rojo ✕
- **Categorías:** puedes agregar o quitar categorías enteras

### 🧴 Productos
- **Agregar producto:** haz clic en "➕ Agregar producto"
- **Precio, nombre, descripción:** edita directamente
- **Destacado:** marca la casilla para que aparezca con una estrella ⭐

### 📸 Galería
- **Agregar foto:** haz clic en "➕ Agregar foto"
- **URL de imagen:** pega el enlace de la foto (ver "Cómo subir fotos" más abajo)
- **Destacada:** las fotos destacadas aparecen primero

### 🎁 Planes de Fidelidad
- Cambia el nombre, descripción y beneficios de cada plan
- Los beneficios se escriben uno por línea

### 📞 Contacto
- Cambia el número de WhatsApp
- Cuando tengas Instagram, agrega el usuario aquí

## 🚀 CÓMO PUBLICAR LOS CAMBIOS

Después de editar todo lo que quieras:

1. Ve a **"Publicar cambios"** en el menú de la izquierda
2. Haz clic en el botón **"🚀 Publicar ahora en GitHub"**
3. Espera el mensaje verde de confirmación ✅
4. Espera **1-2 minutos**
5. Abre la página web para ver los cambios:
   ```
   https://javierandresm13.github.io/peluditos-al-hogar-bogota/
   ```

## 🖼️ CÓMO SUBIR FOTOS (desde celular o PC)

Las fotos NO se suben directamente al admin. Hay que usar un servicio gratis:

### Opción recomendada: ImgBB (gratis, sin registro)

**Desde el celular:**
1. Abre https://imgbb.com en tu navegador
2. Toca **"Start Upload"** o el botón central
3. Selecciona la foto de tu galería
4. En "Embed codes", busca el que dice **"Direct link"**
5. Toca **"Copy"** (copiar)
6. Pega ese enlace en el panel admin (galería o productos)
7. Publica los cambios

**Desde el PC:**
1. Abre https://imgbb.com
2. Arrastra la foto a la página
3. Copia el **"Direct link"**
4. Pégalo en el admin

✅ **Recomendación:** usa fotos cuadradas (1:1) para que se vean mejor.

## ❓ PREGUNTAS FRECUENTES

**¿Puedo editar desde mi celular?**
✅ Sí, el panel admin funciona en cualquier navegador.

**¿Qué pasa si cierro el navegador sin publicar?**
Los cambios quedan guardados en ese navegador. Cuando vuelvas a abrir el admin, estarán ahí. Pero no se verán en la página web hasta que hagas clic en "Publicar ahora".

**¿Cada cuánto toca reconectar el token?**
Nunca. El token se guarda en el navegador y funciona para siempre.

**¿Puedo tener el token en el celular y en el PC?**
✅ Sí, cada dispositivo necesita su propia conexión (solo pegar el token una vez por dispositivo).

**¿Qué hago si la página se ve rara después de publicar?**
Espera 2-3 minutos y refresca (F5 o desliza hacia abajo en el celular). A veces GitHub tarda un poco.

**¿Pierdo las fotos si cambio de celular?**
No, las fotos están en ImgBB. Solo tienes que pegar los enlaces de nuevo en el admin del nuevo dispositivo.
